Procedure:
- Start any one of the machines
- Compile all java files - javac *.java
- Start the other machines mentioned in Configuration file 
- Run 
			1. Main_Server.java for chuck servers - java Main_Server
			2. Main_Client.java for client - java Main_Client
			3. Main_MetaServer.java for Meta servers - java Main_MetaServer
