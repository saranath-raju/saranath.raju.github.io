3. Server_Thread.java
4. Configuration file - consists of the Host names with respective port numbers and the path each process will have to take.
-------------------------------------------------------------------------------------------------
Procedure:
- Start any one of the machines
- Compile all java files - javac *.java
- Start the other machines mentioned in Configuration file 
- Run the Server_main java file  on each of them - java Server_main
- Each process should display its random token value, path taken and sum of all tokens along the path.
