This project includes 3 java files: Listening.java, Server_Main.java and Server_Thread.java and a Configuration file.

The configuration file consists of the Host names with respective port numbers and the path each process will have to take.


1)Initially start one machine - dc01.utdallas.edu
2)Compile all java files - javac *.java
3)Start the other machines mentioned in Configuration file and run the Server_main java file  on each of them - java Server_main
4)Each of the processes should display its random token value, path taken and sum of all tokens along the path.
